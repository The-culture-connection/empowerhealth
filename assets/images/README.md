# Background Images

This directory contains background images used throughout the app.

## Required Images
- `bg1.png` - Background image 1
- `bg2.png` - Background image 2
- `bg3.png` - Background image 3
- `bg4.png` - Background image 4
- `bg5.png` - Background image 5

## Usage
Background images are displayed on screens with a fixed position while content scrolls over them. They are randomly selected using `DS.getRandomBackgroundImage()` from the design system.

Currently, placeholder files exist. Replace these with actual PNG images (recommended size: 1080x1920px or similar portrait orientation) for production use.
