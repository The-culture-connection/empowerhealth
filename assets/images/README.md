# Background Images

This directory contains background images used throughout the app.

## Required Images

Copy these images to this folder:

| Filename | Screen Usage | Notes |
|----------|--------------|-------|
| `Authscreen.jpeg` | Auth, Login, Signup screens | Full-screen with dark overlay |
| `Homeimage.jpeg` | Dashboard/Home screen | Main app screen |
| `babyfamily.jpeg` | Community screen | Community posts feed |
| `family.jpeg` | Feedback screen | Feedback form |
| `helpingheadjpg.jpeg` | Appointments screen | Calendar view |
| `braidinghair.png` | Transcription screen | Recording interface |

## Usage

Each screen uses a specific background image:
- Images are displayed with `BoxFit.cover` to fill the screen
- Fixed position (image stays in place while content scrolls)
- Semi-transparent overlay for better content readability
- Automatic fallback to gradient if image not found

## Image Requirements

- **Format**: JPEG or PNG
- **Orientation**: Portrait (vertical) preferred
- **Recommended Size**: 1080x1920px or higher
- **File Size**: Optimize for mobile (< 1MB per image recommended)
